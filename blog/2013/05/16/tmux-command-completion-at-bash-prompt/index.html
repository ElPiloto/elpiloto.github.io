Tmux Command-completion at Bash Prompt!

May 16th, 2013

After checking out git’s amazing git command-completion and my recent_discovery of how to implement your own command-completion for bash scripts, I began to wonder why tmux didn’t have this feature.  After some searching, I found that the tmux installation includes a script file for command-completion in bash!  In the tmux-1.8 distribution, you should find a folder titled “examples” which contains a file “bash_completion_tmux.sh”.  Just source this file in your .bashrc and you’re done!!
