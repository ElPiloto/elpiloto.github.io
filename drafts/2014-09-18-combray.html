Combray: Memory-guided Perception

Aaron Bornstein, Mariam Aly, Sam Feng

How do we combine the two following pieces of information?

  • behavior guided by hippocampal/episodic memories
  • behavior guided by perceptual evidence

Effects of priors on evidence accumulation

  • random-dot motion task, can modify prior by making a bunch of trials have motion in same direction (i.e. a bunch of leftward moving trials, on the next trial, strong prior for motion towards left)
  • translation of prior to drift-diffusion model:
    possibility #1: modify starting point (usually set to middle, prior shifts it away from middle)
    possibility #2: modify drift rate
    possibility #3: accumulate sensory and memory evidence - this is the one they’re going for

What is neural substrate of the prior?

  • some work showing hippocampus can predict activity of V1
  • other stuff I missed

Alternative 1 for perceptual and memory accumulation: Serial accumulation

Idea: memory starts accumulating, then when perceptual signal comes on stop accumulating memory signal and start accumulating perceptual evidence

Alternative 2 for perceptual and memory accumulation: Parallel accumulation

Idea: sample from memory until percept is available and continue sampling from memory

Task: Probabilistic Sequence Learning

  • hippocampal (and ventral temporal) activity flucturates with difficulty of predicting next item in a sequence (let’s call these variations in difficulty, variations in memory evidence)

Similarly stimuli for scenes (as opposed to faces i.e. f1 = face1)

perceptual noise: rapid flickering

learning trials: perceptual noise is off - participants learn transition structure for that block

test trials: perceptual noise is on - participants should combine their knowledge of the transition structure with perceptual inference to guide decisions
- have either low or high amount of perceptual noise
- high: calibrated to get it right 65%
- low: calibrated to get it right 85% of the time

Pilot 1A/B

Two blocks each with 140/100 learning trials and 40/80 test trials

Test trials: don’t actually show images during test, show white noise

Takeaway: subjects responded “early” 56% of the time (compared to 8.8% during learning) during test phase, seeming to indicate they were relying on memory evidence and realized the test phase didn’t actually have any perceptual evidence because they were shown white noise, not the images

Pilot 2

introduce calibration for perceptual certainty

Takeaway: almost no early responses during test - seem reluctant to use memory evidence

Analysis:
- Do memory and perceptual evidence predict response times? We can do multiple regression with both types of evidence against RT and then look at betas for both memory and perceptual evidence. YES
- Memory evidence for what I ended up choosing decreases:
- high perceptual noise blocks: accumulation threshold increases as a function of memory evidence
- low perceptual noise blocks: accumulation threshold decreases as a function of memory evidence
